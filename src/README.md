# Codigo Fonte

Esta pasta organiza funcoes auxiliares usadas pelo notebook principal.

A logica principal continua no notebook para preservar o formato solicitado no trabalho, enquanto funcoes auxiliares ficam separadas para reduzir repeticao:

- `etl/schema.py`: schema e validacoes.
- `utils/benchmark.py`: medicao de tempo.
- `utils/plotting.py`: configuracao e salvamento de graficos.
